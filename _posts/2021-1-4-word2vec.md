**How to form the training set?**

Denote the training set as $$D$$. For each sentence in our corpus, we set each word in the sentence as the target word. Then, given a context window size $$n$$, we select the left and right context words up to $$n$$ steps from the target word, e.g. for $$n = 2$$, we have:

```
{...} - target word
[...] - window context

the [skip-gram model {predicts} context words] given the target word and CBOW does the reverse.

D = {...,
(predict, skip-gram),
(predict, model),
(predict, context),
(predict, word),
...
}
```

The training set is formed by the (target-word, context-word) pairs with stop and common words being ignored. Mathematically, $$D = (V_t, V_c)$$ where $$V_t$$ and $$V_c$$ are one-hot encodings of dimension $$\lvert V \rvert$$, i.e. the vocabulary size. 

> **Note**: $$V_t$$ and $$V_c$$ are essentially the same domain.

**Skip-gram**

Skip-gram is one of the unsupervised learning techniques used to find the most related words for a given word.

- *Objective Function*: Let the target word embedding be $$w_t$$ and the context word embedding be $$w_c$$, then the closeness $$u_c$$ between $$w_t$$ and $$w_c$$ can be computed by *dot-product* such that $$u_c = w_c \cdot w_t$$. The objective is to maximise the conditional probability of the context words given the target word:

$$
P(\text{context} | V_t) = P(V_{t-n}, ..., V_{t-1}, V_{t+1},..., V_{t+n} | V_t) = \prod_{k \in \mathbf{Z} \cap [-n, n] \setminus \{0\}} P(V_{t+k} | V_t) 
$$
