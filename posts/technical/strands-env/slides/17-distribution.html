Distributed Rollouts

To scale rollouts past Python's GIL, we can use processes — e.g., Ray actors: 

@ray.remote                          # its own process: own GIL, own event loop
class EnvironmentActor:
    async def step(self, action):
        return await self.env.step(action)

pool = EnvironmentActorPool(n_actors_per_node=8)   # 8 per node, across the cluster
result = await pool.step(action)                   # round-robin to a free actor
✓ Each actor is a separate process — async inside each, parallel across. Work that used to pile up in one event loop now runs across many actors.
